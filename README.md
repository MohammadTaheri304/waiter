## waiter

### Prerequisites to develop and test project
- JDK 11 (JavaSE 11)
- Gradle 5.3+

### How to build?
Check prerequisites, then run:
```
./gradlew clean installDist
```

### How to test?
```
./gradlew clean test
```

### Javadoc
To generate javadoc, run:
```
./gradlew javadoc
```
